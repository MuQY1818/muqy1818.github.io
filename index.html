    Module 03
    ECCV 2026 Co-first Author

    RoboStream: Weaving Spatio-Temporal Reasoning with Memory in Vision-Language Models for Robotics

    Yuzhi Huang*, Jie Wu*, Weijue Bu*, Ziyi Xiong, Gaoyang Jiang, Ye Li, Kangye Ji, Shuzhao Xie, Yue Huang, Chenglei Wu, Jingyan Jiang, Zhi Wang (* equal contribution)

    A training-free framework for long-horizon robotic manipulation that integrates spatio-temporal reasoning with persistent memory. By fusing visual evidence with 3D geometry (STF-Tokens) and maintaining a Causal Spatio-Temporal Graph (CSTG), RoboStream enables persistent object grounding and occlusion recovery — 90.5% success on RLBench, significantly outperforming baselines on challenging real-world tasks.

    Under Review Co-first Author

    ChainVLA: Chaining Task Progress and Motion Intent Across VLA Queries

    Weijue Bu*, et al. (* co-first author)

    Receding-horizon VLA policies replan every few steps, but the process does not by itself transfer the execution state behind one query to the next. ChainVLA makes this handoff explicit as a two-timescale execution-state transition: Progress Context combines a recurrent Working State with evidence retrieved from a Sparse Event Chain, while Motion Tail carries the preceding horizon's unfinished suffix as a revisable continuation prior. ChainVLA achieves 62.8% average success on RMBench and 98.8% average across four LIBERO suites.

    IEEE TCC 2026 Student First Author

    CALO: Code-Aware and Load-Aware Resource Configuration for Serverless Functions via Deep Reinforcement Learning

    Donghong Xu, Weijue Bu, Zhouliang Ye, Shilong Wu

    CALO is a code-aware and load-aware DRL controller for serverless resource configuration. It caches offline code embeddings and fuses them with live telemetry in a 79-dimensional state, selecting memory, architecture, and timeout from a 48-action provider catalog. Trained in a measurement-grounded simulator calibrated on OpenWhisk profiles, CALO improves mean reward from 0.513 to 0.569 and CVaR10 from 0.163 to 0.351 over online Bayesian optimization, with lightweight online inference (0.124 ms median).

    ICME 2026 First Author

    Conscious Gaze: Adaptive Attention Mechanisms for Hallucination Mitigation in Vision-Language Models

    Weijue Bu, Guan Yuan, Guixian Zhang

    Large Vision-Language Models often exhibit text inertia, where attention drifts from visual evidence toward linguistic priors, causing object hallucinations. Conscious Gaze (CG-VLM) is a training-free, inference-time framework that converts game-theoretic interpretability into decoding control: a Cognitive Demand Sensor built on Harsanyi interactions detects when visual grounding is needed, and Focused Consensus Induction reorients mid-layer attention toward visual tokens. CG-VLM achieves state-of-the-art results on POPE and CHAIR across InstructBLIP, LLaVA, Qwen-VL, and mPLUG.

    PiN-Mod: Identity-Consistent Personalized Image Generation

    Weijue Bu, et al.

    PiN-Mod is a framework for personalized text-to-image generation that balances identity fidelity and semantic alignment via identity-guided noise modulation and a dual-prompting strategy. We also release PortraitEditX, a large-scale multimodal dataset for benchmarking identity consistency and cross-modal control.
